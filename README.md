# 开源仿MT管理器的文件管理器项目

>开发初衷:就是想仿一个类似MT管理器的项目，看了一下关于双列表的文件管理器的开源项目几乎没有，自己也学习了一阵子的安卓开发，就想着能不能写一个这样的项目来练练手

## 目前实现功能:

### 1.文件双列表和基础功能UI (完成)

### 2.文件双列表滑动滑块 (完成)

### 3.下拉刷新动画和列表加载动画 (完成）

### 4.列表阴影折叠效果控件 (完成）

### 5.左右滑动选中文件/文件夹 (完成）

### 6.利用零宽获取Android/data文件夹 (完成）
